# wireframe_de_media_fidelidade
1° Desafio do Bootcamp DIO Formação UI/UX Designer.

Descrição do Desafio

O desafio envolve a criação de um wireframe de média fidelidade para uma única página (One Page) de algum produto à sua escolha. Esse produto pode ser qualquer coisa que você queira apresentar, como um aplicativo, uma página de vendas, ou um produto de aula, por exemplo.

Utilização da ferramenta Figma.

Criei uma página com diversos tipos de plano de kit de produtos e suplementos para quem busca manter uma vida fitness pré e pós treino recebendo em casa uma caixa surpresa com quantidade de produtos de acordo plano escolhido.
